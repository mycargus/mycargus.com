# Mikey Hargiss's Personal Website

[mycargus.com] is a [Next.js] project bootstrapped with [`create-next-app`].

## Get Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the
result.

## Deploy

This project is hosted on [Vercel]. Automated deployments for both pull-requests
and changes to the `main` branch are managed through the Vercel --> Github
integration.
